A simple lite weight plugin for use with the theme from SwiftThemes.Com

== Description ==

A very simple and super lite slider plugin with only 5kb of JS & CSS. Plugin features a clean and easy UI with live preview to create infinite number of sliders that are search engine optimized.


== Installation ==


1. Upload the plugin files to the `/wp-content/plugins/simple-slider` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress.
1. Create new sliders and insert them anywhere using the shortcode.


== Frequently Asked Questions ==

= Is it only for themes released by SwiftThemes.Com =
You can use the slider with any theme.
